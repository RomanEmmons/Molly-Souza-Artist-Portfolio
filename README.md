# Molly Souza — Artist Portfolio

A professional portfolio website built for Molly Souza, a visual artist. The site provides a clean, focused space to showcase her work and make it easy for visitors to explore her portfolio and get in touch.

## About the project

This was a client project — designed and built end-to-end from requirements through deployment. The goal was a lightweight, fast-loading site that puts the artwork front and center without getting in the way.

## Tech stack

- **Frontend:** JavaScript, HTML, CSS
- **Bundling:** Webpack

## Getting started

**Install dependencies:**
```bash
npm install
```

**Run in development:**
```bash
npm run dev
```

**Build for production:**
```bash
npm run build
```

## Project notes

This was a real client deliverable, not a personal project. Requirements were gathered directly from the client, design decisions were made collaboratively, and the final site was deployed for public use. The focus was on simplicity and performance — an artist's portfolio should load fast, look great, and stay out of the way of the work.
